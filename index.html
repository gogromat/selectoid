Selectoid is a JavaScript plugin that turns mean select boxes into happy ones. Selectoid was originally inspired by Google Translate page's select box. However selectoid has its limitations - it likes small data set - up to a 50~80 elements. After that, feel free to use any of the select box enhancing JS libraries with auto-complete capabilities that are out there - Chosen, Select2, etc.

Before we continue: there is 10 pixel padding on this page for representational purposes. Don't be lost when it says "100% width div" and it looks 20 pixels short.

Feature: Selectoid based on jQuery object:
  • useful when you auto-generate Selectoid input fields
  • useful when you do not have an id of the element you want to make into a Selectoid
  • is an alternative way to make element/field into a Selectoid
Example:
  • both divs have same class "someClass"
  • we create a new Selectoid with object: $($(".someClass")[0]) property in parameters
  • only first item should turn into a Selectoid
  • first item should now have a unique Selectoid id - it is auto-assigned if item had no id before
  • first item (element with class "someClass") now has an id selectoid_0
  • first item closes on mouseLeave action

In this example we change the data of the Selectoid that was initialized previously with completely new data. Initially data should be languages, but later on it is changed to cities.
